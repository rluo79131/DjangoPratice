## 說明
- 使用 Django 框架技術，來實作簡易的網站功能。
- 採用 MySQL 作為資料庫方案。

## 運行指令
- 安裝套件：
pip install django
- 建立專案：
django-admin startproject [project]
- 建立應用程式：
python manage.py startapp [app]
- 資料遷移：
python manage.py makemigrations [app]
python manage.py migrate [app]
- 運行程式：
python manage.py runserver
